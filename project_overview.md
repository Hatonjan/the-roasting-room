**Phase 0: Foundation & Planning (What we'll do first)**

**Goals:**

* Design database schema
* Understand relationships in e-commerce context
* Set up project structure
* Configure Git workflow with branches

**Key Learning:**

* Database normalization
* Entity-Relationship diagrams
* Professional project organization

**Phase 1: Backend Foundation (Django + PostgreSQL)**

**Goals:**

* Set up Django with PostgreSQL
* Create models (Product, Category, User extensions)
* Configure Django admin
* Understand Django ORM

**Key Learning:**

* Django project vs app concept
* Model design and migrations
* PostgreSQL in Django
* Admin customization

**Phase 2: Authentication System**

**Goals:**

* Implement user registration
* Create login/logout endpoints
* JWT token generation
* Understand token-based auth

**Key Learning:**

* Django REST Framework basics
* Authentication vs Authorization
* Token storage strategies
* Security best practices

**Phase 3: Product API**

**Goals:**

* Create product endpoints (CRUD)
* Implement permissions (public read, admin write)
* Add filtering and search
* API documentation

**Key Learning:**

* REST API design principles
* Serializers in DRF
* Permissions and ViewSets
* API testing with Thunder Client

**Phase 4: Frontend Setup (React)**

* Create React app with Vite

* Design component architecture
* Configure API connection

**Key Learning:**

* Modern React project structure
* Environment variables in React
